# 節目組數據工具

鏡好聽（Mirror Voice）節目組內部使用的數據分析工具。純前端、無伺服器，所有資料在使用者瀏覽器本機處理，不會上傳到任何地方。

## 工具

- **RSS節目收聽分析**（`tool-1/`）：分析節目在 Apple Podcast、Spotify、YouTube 三大平台的收聽表現。
- **鏡好聽平台收聽分析**（`tool-2/`）：分析鏡好聽 App 每週收聽 log。

## 使用方式

開啟 `index.html`（或部署後的網址）即可使用，不需安裝。

## 給維護者 / AI

修改前請先讀 `CLAUDE.md`（＝`維護指南.md`），裡面有所有規則、已知地雷與版本演進。

## 注意

本 repo 只放工具程式碼。**任何節目收聽的 CSV／xlsx／log 資料檔都不應上傳到此處。**
