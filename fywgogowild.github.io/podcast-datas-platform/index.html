使用提醒
  • 所有工具的資料邏輯不同,數字不能直接相加或比較(例如 Apple 的「播放次數」跟鏡好聽的「播放事件」定義不一樣)
  • 所有資料解析、計算都在你的瀏覽器本機完成,檔案不會上傳到任何伺服器。唯一例外:RSS工具的「上傳摘要到彙整表」功能,由製作人主動啟用後,只把報表摘要數字(非原始檔)送到節目組自己的 Google 試算表
  • 產出的報表都可以下載成獨立 HTML 檔,方便分享
  • 各工具版本各自記錄(見上方卡片),完整更新紀錄在 CHANGELOG
